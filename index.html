ShardMC - Your Minecraft server core.

ShardMC is an open-source, Minecraft Server Core Software, designed to be customizeable and very fast!

Downloads
Learn more
Note: the project is not production ready yet! It's better to wait for the v1.0, before using it in real environment!

Features

What you get with ShardMC

ShardMC has a lot of cool features designed for players by players!

Customizeable

Modules allow you to customize the server like you want! Like plugins, but better... and the best part is: modules can add support for your favourite Mod Loaders!

Fast

ShardMC is designed with speed in mind! We optimized the heavy calculations and some other stuff!

Open to new ideas

We are open to communitiy's ideas! If you know how to make this project better: contact us on Discord!

Open Source

ShardMC is as open-source as it can be! Contributions are welcome :D

Roadmap

Echo v0.1

First release of Echo - bytecode manipulation library for ShardMC, that allows to create modules and modify server's code!

Amethyst v0.1

With the first release of Amethyst, it would be possible to launch the core and attach modules!
